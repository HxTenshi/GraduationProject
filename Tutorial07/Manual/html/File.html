プロジェクトの構成

+ Assets //ゲームのリソースフォルダ

-リソース

+ EngineResource //てんしエンジンのリソースフォルダ

-リソース

+ ScriptComponent //スクリプトのプロジェクトフォルダ

　　- Tutorial07_2010.sln //スクリプトのソリューション

+ Scenes //ゲームシーン一覧のフォルダ

　　+ xxxx //ゲームシーンのフォルダ

　　　　- yyyy.json //ゲームオブジェクトファイル

- Tutorial07.exe //てんしエンジン実行ファイル

てんしエンジン独自の拡張子

tesmesh : TenshEngineStaticMesh アニメーションしない頂点とインデックデータ

tedmesh : TenshEngineDynamicMesh アニメーション用にウェイト情報付き頂点とインデックデータ

tebone : TenshEngineBone ボーンのデータ

scene : ゲームシーンのオブジェクト群のデータ

json(仮) : ゲームオブジェクトのデータ

txt(仮) : マテリアルのデータ
