Script

Scriptの概要

ゲームオブジェクトにユニークな機能やギミックをC++言語で独自に作成することが出来ます。

Scriptの作成方法と導入

1.プロジェクトフォルダの「./ScriptComponent/Tutorial07_2010.sln」を開きます。

2.ScriptComponentプロジェクトのScript.hに「IDllScriptComponent」を継承したクラスを作成します。

3.ScriptComponentプロジェクトのScript.cppのFuctorySetterクラスのコンストラクタに_ADD( 作成したクラス );を追加します。

4.てんしエンジンでスクリプトのコンパイルボタンを押してコンパイルに成功すれば、Sctriptの作成完了です。

5.Scriptを導入したいゲームオブジェクトにScriptComponentを追加してClassのパラメーターに作成したScriptのクラス名を入力すれば導入完了です。

IDllScriptComponentの説明

継承できる関数の説明

void Initialize() : ゲーム開始時に呼ばれます。

void Start() : ゲームシーンのオブジェクト全てのInitializeが呼ばれた後に呼ばれます。

void Update() : 毎フレーム呼ばれます。

void Finish() : ゲーム終了時に呼ばれます。

void OnCollide(Actor* target) : PhysXとPhysXColliderコンポーネントを導入している時に別のPhysXとPhysXColliderコンポーネントを導入しているゲームオブジェクトにぶつかった時に呼ばれます。

引数 Actor* target : ぶつかった相手のゲームオブジェクト。

変数の説明

Actor* gameObject : Scriptを導入しているゲームオブジェクトのポインター。（関数の呼び出し元）

IGame* game : ゲームシーンのポインター。ゲームオブジェクトの生成、削除、検索などが行える。

Inspector上にパラメーターの追加方法

1.プロジェクトフォルダの「./ScriptComponent/Tutorial07_2010.sln」を開きます。

2.ScriptComponentプロジェクトのReflection/ReflectionSetter.cppを開きます。

3.ReflectionSetterクラスのコンストラクタにREF( 作成したクラス , 表示する変数 );を追加します。

4.てんしエンジンでスクリプトのコンパイルボタンを押してコンパイルに成功すれば、完了です。

注意

まだ動作チェックを行っていない事が多いので何が起こるか分からない。

Scriptから呼ぶことを想定されていないのに呼べてしまう関数などもあるはず。
